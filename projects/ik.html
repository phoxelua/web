Inverse Kinematics

CS184 Graphics

This was built for project 4 in CS184 at UC Berkeley. Built using: C++

Intro

We animated a virtual skeleton arm to move along a path using C++ via inverse kinematics. Inverse kinematics basically takes in a point in space and calcuates the rotations neeeded to move the arm to the given point (as opposed to forward kinematics which uses user specified angles to move a joint). The rotations were calculated using a numerical method consisting of calculating the pseudo inverse of the Jacobian matrix. Out of range targets were also handled along with arbitrary number of joints and length.
