EverythingPersonalSchoolOther

Tron Wannabe


A simple TRON-inspired, javascript single player game with music from the movie. Very basic computer AI.

Sharks and Fish


Data structures CS course. Simulated an ocean full of sharks and fish. Also implemented a run-length encoding representation of the ocean.

Yelp Wannabe


Too lazy to go all the way to your browser and lookup Yelp reviews? No? Oh well I was. So, I built a command prompt application that scrapes Yelp for restaurant reviews. It also reads the reviews and tries to analyze them for the overall attituded.

Lame Dictionary


First project with web scraping! Looks up definitions and parts of speech from the command prompt. Very basic, nothing special. Currently implementing an enhanced thesaurus function.

Remote Controlled Laptop


Uses simple batch programming with basic VBA scripts to control my laptop with SMS messages. Currently has basic modes such as opening the browser to my favorite tabs and playing music.

ECG


Working electrocardiogram (ECG) that accurately measures heart rate. Utilizes chains of low pass filters and op-amps to clean up the signal but addional open source software was used to get a better-looking reading.

Biofilters


Cheap biofilters made of woodchips, gravel, and compost to filter nitrates and phosphates out of agricultural run-off. Varying ratios of these three biomedia were tested to determine the optimal ratio for run-off reduction. Pictures coming soon!

Shoe Rack


Challenge (self-imposed): build a shoe rack under $30 using as much scavenged materials as possible. Take at most 2 weeks.

Uncle's House


I helped my uncle build his house by installing the foundation and floor for the catwalk in the attic as well as installing the roof rafters and shingles. I also cut and created the header frames for the arches in the living room and kitchen door.

The Game of Hog


Introductory CS course. Developed a simulator and multiple strategies for the dice game Hog. Practice with higher order functions and probability analysis (self-imposed criteria).

Twitter Trends


Introductory CS course. Developed a geographic visualization of Twitter data across the US. Practice with dictionaries, lists, abstraction, and modular programming.

"Catapult" Thing


Bored? Have the afternoon free? Then why not build a catapult? But what happens when a MechE and EECs amateur try to build a catapult on the spur of the moment? Hint: a very, very jank catapult.

Ants vs. SomeBees


Introductory CS course. Created a tower defense game. Practice with OOP and inheritance. Most skeleton code provided.

Scheme Interpreter


Introductory CS course. Implemented a scheme interpreter for the Scheme language. Practice with parsing, tokenizing, and evaluating Scheme functions.

Network (The Game)


Data structures CS course. Implemented a program that plays the game Network against a human player and another computer program. Practice with game strategies and alpha-beta pruning. Honestly, the hardest project for me ever (so far).

WUGs and Trees


Data structures CS course. Augmented a variety of data structures to achieve faster performance with representations of weighted directed graphs and min span trees. Practice with Kruskal's algorithm.

MapReduce in EC2


Machine structures CS course. Used EC2 to calculate co-occurence rates of various words in large text files.

Lame Mail


Just another command prompt email checker (a Mutt rip-off). One of my first python programs. Mainly implemented to replace Outook and have RC Laptop run independently.

Desk


Built a very cheap (and very jank) side desk for a friend. Couldn't scavenge my way out of this one since there were no large pieces of wood to be found.

Phong Shading

MIPS Simulator

Created an instruction interpreter for a subset of MIPS assembly code. This included 18 R-Types, 2 J-types, and 17 I-type instructions.

Image Processing and Optimization

Optimized 2D image convolution by implementing SSE intrinsics, loop unrolling, cache blocking, register blocking, and OpenMP. Was able to achieve an average of 68 Gflop/s for images ranging from 240 x 240 to 1200 x 1200. Credits of image goes to Ajay Tripathi.

Processor Design

Used Logism to implement a 16-bit, 2-stage pipelined processor that executes MIPS assembly. Deliverables included register files, ALU, processor, memory, and output devices.

Pacman AI

A 5 part project that implemented a Pacman AI using: heuristics, search and Markov decision making, reinforcement learning, and probability.
