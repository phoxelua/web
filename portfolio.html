Flowbit, Software and Electrical Enginering Intern
7/2013 - 8/2013

I worked with a team of three other engineers to design, program, and test an Arduino-based prototype that remotely monitors water usage in Third World countries. On the hardware side, I designed and integrated the Arudino with GPS shield, GPRS shield, RTC, and flow meter. On the software side, I programmed the Arduino to send SMS, receive GPS information, and process flow meter data. Projects

Hardware / Software

Yelp Wannabe

I built a command prompt application that scrapes Yelp for the user's search query and preferences. The version 1.0 only scraped for price, ratings, and number of reviews. Version 2.0 analyzed the comments and attempted to find the overall attitude in each of the reviews. Version 3.0 usuccessfully attempted to thread each review analysis to processes in parallel but this showed no significant improvement in run time.

Lame Dictionary

I built a web scraper that looked at dictionary websites for basic definitions. This was my first project with webscraping. I am currently implementing a thesaurus that looks for synonyms based on intensity (positiveness / negativeness), usage, and context (professional / colloquial)

Remote Controlled Laptop

I used simple batch programming with basic VBA scripts to control my laptop with SMS messages. RC Laptop just has just has basic modes such as opening the browser and playing music. Intially, I used Outlook to trigger rules but I decided to replace it with my Lame Mail application.

Lame Mail

I built a command prompt based email checker (a Mutt rip-off) as a way to learning how to use the python url library. It was one of my first python programs.

ECG

I designed and built the a working electrocardiogram (ECG) to accurately measure my heart rate. It used chains of low pass filters and op-amps to clean up the signal but addional open source software was used to get a better-looking reading.

Biofilters

I designed cheap biofilters made of woodchips, gravel, and compost to filter nitrates and phosphates out of agricultural run-off. Varying ratios of these three biomedia were tested to determine the optimal ratio for run-off reduction. The projet place 1st at the Greater San Diego Science and Engineering Fair and eventually made it to the California State Science Fair.
