Super Cool Typing Game

Some friends and I made a 2 player typing game for BearHack, a 24 hour hackathon. Made using node.js, socket.io, and angular.js.

CaptchaMeNot

CaptchaMeNot was built by me and two other people mainly with MATLAB to read very simple Captchas for HackJam, another 24 hackathon hosted at UC Berkeley. It identifies characters by projecting samples into a histogram that can then be mapped to a dictionary of candidates.

View Finder

Built by my roomate and I for Facebook Hackathon. ViewFinder is a geography-based image viewer that pulls images from Instagram in real-time and displays them in an river / carousel-like fashion. Just pick a location and it will show you all the pictures tagged from that location!

Mini Tron

To learn Javascript, I made a game based on the old arcade classic TRON in pure Javascript.

Yelp Wannabe

I built a command prompt application that scrapes Yelp for the user's search query and preferences. The version 1.0 only scraped for price, ratings, and number of reviews. Version 2.0 analyzed the comments and attempted to find the overall attitude in each of the reviews. Version 3.0 usuccessfully attempted to thread each review analysis to processes in parallel but this showed no significant improvement in run time.

Lame Dictionary

Remote Controlled Laptop
